# 电影评论网站项目结构说明

## 项目概述
这是一个基于Vue3技术栈的电影评论网站，实现了所有必需的功能模块。

## 完整项目结构
```
FilmReviewWebsite/
└── my-vue-app/                 # Vue3项目根目录
    ├── node_modules/          # 依赖包目录
    ├── public/                # 静态资源目录
    │   └── favicon.ico
    ├── src/                   # 源代码目录
    │   ├── assets/            # 静态资源
    │   │   └── vue.svg
    │   ├── components/        # Vue组件
    │   │   └── HelloWorld.vue

    │   │   ├── MovieListView.vue # 电影列表页
```

## 技术实现细节

### 1. Vue Router 实现
- 文件: `src/router/index.js`
- 定义了5个核心页面路由：首页、电影列表、电影详情、登录、注册
- 使用 createWebHistory 模式

### 2. Pinia 状态管理 实现
- 文件: `src/stores/index.js`
- 管理用户登录状态、电影数据、评论数据
- 包含登录、注册、登出、添加评论等操作方法

### 3. UI 框架 (Element Plus) 使用位置
- 在 `src/main.js` 中全局注册
- 在各页面中使用了以下组件：
  - [el-carousel](./src/views/HomeView.vue#L3-L8) / [el-carousel-item](./src/views/HomeView.vue#L4-L7) (轮播图) - HomeView.vue
  - [el-input](./src/views/MovieListView.vue#L7-L13) (搜索框) - MovieListView.vue
  - [el-button](./src/views/MovieDetailView.vue#L28-L35) (提交按钮) - MovieDetailView.vue
  - [el-form](./src/views/LoginView.vue#L7-L24) / [el-form-item](./src/views/LoginView.vue#L8-L23) (登录表单) - LoginView.vue 和 RegisterView.vue
  - [el-menu](./src/App.vue#L12-L24) / [el-menu-item](./src/App.vue#L15-L16) (导航菜单) - App.vue

### 4. Axios 数据请求 使用位置
- 文件: `src/utils/api.js`
- 定义了获取电影、提交评论、登录、注册等API接口（使用模拟数据）

### 5. 评论模块 实现
- 在 `src/views/MovieDetailView.vue` 中实现
- 仅对登录用户显示评论表单
- 使用Pinia存储和管理评论数据
- 包含评论验证和日期格式化功能

### 6. 登录注册功能 实现
- 登录: `src/views/LoginView.vue`
- 注册: `src/views/RegisterView.vue`
- 使用Pinia管理用户状态
- 将用户信息存储在localStorage中

## 页面功能说明

1. **首页** (`HomeView.vue`)
   - 轮播图展示热门电影
   - 热门电影推荐网格
   - 响应式布局

2. **电影列表页** (`MovieListView.vue`)
   - 电影网格展示
   - 搜索过滤功能
   - 电影信息展示

3. **电影详情页** (`MovieDetailView.vue`)
   - 电影详细信息展示
   - 评论模块（仅登录用户可评论）
   - 评论列表展示

4. **登录页** (`LoginView.vue`)
   - 用户名密码验证
   - 表单验证
   - 登录状态管理

5. **注册页** (`RegisterView.vue`)
   - 用户注册表单
   - 表单验证（用户名、邮箱、密码确认）
   - 注册状态管理
